Engine Overview

The Basics

The Exokit Engine is a native 3D XR web engine.

First principles

The Exokit Engine is built using Node.js. It is primarily used to power 3d experiences, but can emulate 2D. For example, emulating a 2D web browser based on Chrome Embedded Framework to use for 2D interfaces.

Exokit Engine and the Web

Exokit Engine provides native hooks for WebGL, WebXR, WebVR, WebAudio, and other APIs used for immersive experiences. The Exokit Engine loads web pages and experiences built with your favorite frameworks like THREE.js, A-Frame, Babylon.js, etc.
